Linux Basics

Introduction

Linux is a powerful operating system widely used in servers, cloud environments, and cybersecurity.

ls

Lists files and directories in the current location.

Example

ls

Output

Shows files and folders in the current directory.

---

pwd

Prints the current working directory.

Example

pwd

Output

Displays the full path of the current directory.

---

cd

Changes the current directory.

Example

cd Documents

Output

Moves to the Documents directory.

---

What I Learned

- Linux is case-sensitive.
- Commands are executed through the terminal.
- Navigation is an important Linux skill.
- "ls", "pwd", and "cd" are basic commands used daily.
